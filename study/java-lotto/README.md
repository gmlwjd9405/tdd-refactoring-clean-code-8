## java-lotto 학습 내용

### 목표
- TDD 기반으로 프로그래밍하는 경험
- 메소드 분리 + 클래스를 분리하는 리팩토링 경험
  - 로또 구현을 통해 단위 테스트, 리팩토링, 객체 지향 프로그래밍 연습을 한다.
  - 클래스, 메서드 이름을 통해 의도를 드러내 다른 사람이 읽기 좋은 코드를 구현하는 경험을 한다.
- 점진적으로 리팩토링하는 경험
